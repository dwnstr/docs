# In-Game Usage

![](https://i.imgur.com/DwIvplS.png)

ULC includes an intuitive menu with easy-to-use controls to change various settings. To access the menu use the ``/ulc`` command!

![](https://i.imgur.com/Y1PgIZG.png)

## Commands

| Command     | Effect                                                   |
| ----------- | -------------------------------------------------------- |
| `/ulc`      | Displays the settings menu and enables dragging the HUD. |
| `/ulcReset` | Wipes all user-preferences and sets default values.      |

## Key-binds

Default key-bindings are ``NUM 1`` - ``NUM 9``. You can change these in your game settings.
